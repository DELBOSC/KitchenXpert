### How do I reset my password?

If you've forgotten your password:

3. Enter your registered email address

**Didn't receive the email?**

## Kitchen Design

### How do I start a new design?

You have three flexible ways:

#### Method 1: Blank Canvas

1. Dashboard > "New Blank Design"
2. Enter kitchen dimensions (width, depth, height)
3. Choose room shape (rectangular, L-shaped, U-shaped, custom)
4. Add doors and windows
5. Start adding components

#### Method 2: AI Questionnaire

1. Dashboard > "AI Questionnaire"
2. Answer 15-20 questions about:
   - Kitchen dimensions
   - Cooking habits
   - Storage requirements
   - Appliance preferences
   - Style and budget
3. Submit and wait 5-15 seconds
4. Review AI-generated design
5. Customize as desired

#### Method 3: Import Floor Plan

1. Dashboard > "Import Floor Plan"
2. Upload image (JPEG, PNG, PDF - max 25MB)
3. Set scale using known dimension
4. Trace walls, doors, windows
5. Begin designing

See [First Design Tutorial](tutorials/beginner/first-design.md) for step-by-step
guidance.

---

### Can I import my floor plan?

**Yes!** Supported formats: JPEG, PNG, PDF (max 25MB)

**Import process:**

1. Prepare clear, well-lit image of blueprints
2. Upload via Dashboard > Import Floor Plan
3. Set scale by clicking two points with known distance
4. Enter actual measurement (e.g., "3.5 meters")
5. Trace walls, doors, windows with tracing tools
6. Start designing

After proper scaling, measurements are accurate within ±2cm.

**Tips:**

- Use plans with at least one known dimension
- High-contrast images work best
- If you have CAD files (.dwg, .dxf), export to PDF first

## Technical Issues

### Why is my 3D view not loading?

**Common causes and solutions:**

#### 1. Browser Compatibility

**KitchenXpert requires WebGL support.**

Check: Visit [get.webgl.org](https://get.webgl.org) - should show spinning cube

**Enable WebGL:**

- Chrome/Edge: `chrome://flags` > Search "WebGL" > Enable > Restart
- Firefox: `about:config` > Search `webgl.disabled` > Set to `false`
- Safari: Preferences > Advanced > Show Develop menu > Develop > Enable WebGL

#### 2. Graphics Card Issues

Update graphics drivers (Nvidia, AMD, Intel). Ensure laptop uses dedicated GPU,
not integrated.

**Minimum:** GPU with WebGL 2.0, 2GB video memory, OpenGL 3.3+

#### 3. Browser Cache

**Clear cache:**

- Chrome: Ctrl+Shift+Delete > "Cached images and files" > "All time" > Clear
- Firefox: Ctrl+Shift+Delete > "Cache" > Clear
- Safari: Preferences > Privacy > Manage Website Data > Remove kitchenxpert.com

Close browser completely, reopen, and log in.

#### 4. Ad Blockers / Extensions

Test in incognito mode (Ctrl+Shift+N). If works, an extension is interfering.
Whitelist kitchenxpert.com in ad blocker.

#### 5. Network Issues

Corporate/school networks may block WebGL. Try home network or mobile hotspot.

#### 6. Performance

If slow but loads:

- Settings > Graphics > Quality: Low/Medium
- Disable shadows
- Simplify design (hide layers)
- Close other tabs

**Still not working?** Help > Generate Diagnostic Report > Email to
support@kitchenxpert.com

**Workarounds:** Use 2D view (Space key), try different browser, use mobile app.
