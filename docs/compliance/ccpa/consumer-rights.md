## Introduction

This document details how California consumers can exercise their rights under
CCPA.

**Contact**: privacy@kitchenxpert.com | 1-800-KX-PRIVACY |
https://kitchenxpert.com/privacy/request

---

## Right to Know

### Categories Request

Consumers can request:

- Categories of personal information collected
- Categories of sources
- Business purpose for collection
- Categories of third parties receiving data

### Specific Pieces Request

Consumers can request actual personal information collected about them.

### Response Format

JSON, CSV, or PDF format delivered via secure download link.

---

## Right to Delete

### What Can Be Deleted

- Account information
- Profile data
- Design data
- Marketing preferences
- Analytics data

### Exceptions

We may retain data for:

- Completing transactions
- Security and fraud prevention
- Legal compliance
- Internal analytics (anonymized)

### Process

1. Submit deletion request
2. Verify identity
3. Confirm scope
4. Process deletion within 45 days
5. Notify third parties

---

## Right to Opt-Out of Sale

### How to Opt-Out

**Do Not Sell Link**: https://kitchenxpert.com/privacy/do-not-sell

**Global Privacy Control**: Automatically honored

**Cookie Settings**: Disable advertising/marketing cookies

### What Changes

- Advertising cookies disabled
- Third-party pixels removed
- Affiliate tracking disabled
- Partner marketing data sharing stopped

### Authorized Agents

Consumers may designate authorized agents with written permission.

## Right to Non-Discrimination

### Our Commitment

Consumers exercising CCPA rights receive:

- Same service quality
- Same prices
- Same product availability
- No retaliation

### Financial Incentives

If offered, financial incentive programs are:

- Voluntary opt-in
- Clearly disclosed
- Reasonably related to data value

---

## Right to Correct

### What Can Be Corrected

- Account information (name, email, address)
- Profile preferences
- Design metadata
- Order information (with verification)

### Process

1. Submit correction request
2. Specify inaccurate data
3. Provide correct information
4. Verification
5. Correction within 45 days

---

## Verification Process

### Verification Levels

| Request Type       | Verification Required   |
| ------------------ | ----------------------- |
| Categories request | Reasonable verification |
| Specific pieces    | High verification       |
| Deletion           | High verification       |
| Correction         | High verification       |
| Opt-out            | Low verification        |

### Verification Methods

**Reasonable**: Email confirmation from registered account

**High**: Email + security questions OR email + government ID

### Authorized Agents

- Written authorization required
- Agent identity verification
- Consumer identity verification (unless power of attorney)

## Response Timelines

### Standard Timeline

| Action                 | Timeline   |
| ---------------------- | ---------- |
| Receipt acknowledgment | 10 days    |
| Verification           | 10-15 days |
| Response               | 45 days    |
| Extension (if needed)  | +45 days   |

### Extensions

We may extend the response period by 45 days when:

- Request is complex
- Multiple requests received
- Additional verification needed

Consumer is notified of extension within initial 45-day period.

### Free Requests

Consumers may make requests twice per 12-month period free of charge.
