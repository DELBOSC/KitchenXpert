USAGE INSTRUCTIONS
================================================================================

QUICK DEPLOYMENT:
  sudo ./deploy.sh dev      # Development
  sudo ./deploy.sh prod     # Production

MANUAL DEPLOYMENT:
  1. Copy files: sudo cp -r . /etc/nginx/
  2. Enable sites: cd /etc/nginx/sites-enabled && sudo ln -s ../sites-available/*.conf .
  3. Test: sudo nginx -t
  4. Reload: sudo systemctl reload nginx

ENABLE SPECIFIC SITE:
  cd /etc/nginx/sites-enabled
  sudo ln -s ../sites-available/api.conf .
  sudo nginx -t && sudo systemctl reload nginx

VIEW LOGS:
  sudo tail -f /var/log/nginx/error.log
  sudo tail -f /var/log/nginx/api.access.log

TEST CONFIGURATION:
  sudo nginx -t

RELOAD NGINX:
  sudo systemctl reload nginx

TO ADJUST CACHING:
  Modify cache settings in nginx.conf and virtual host configs

TO ENABLE FEATURES:
  Uncomment relevant sections in configuration files
  Examples: HTTP/2, client certificates, basic auth, IP whitelisting
